Edit your post function

To edit a post function with configuration, you need a code snippet that contains the url of the edit page (this one)


"edit": {
      "url": "/edit?id=\{{postFunction.id}}&config=\{{postFunction.config}}"
}

This page will probably look similar to the create page, but you will probably want to include the configuration of the workflow to initialise your page's state. As with the create page, you will also need to use JavaScript to validate and persist configuration data.


AP.require(["jira"], function(jira) {
  // When the configuration is saved, this method is called. Return the values for your input elements.
  jira.WorkflowConfiguration.onSave(function() {
    return "string for JIRA to store";
  });

  // Validate any appropriate input and return true/false
  jira.WorkflowConfiguration.onSaveValidation(function() {
    return true;
  });
});
